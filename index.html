This is my developer story. I never had access to or knowledge of computers as a child. I was only permitted to use the computer in high school for homework purposes, and I could only do so once a day for a nominal charge. I learned how to use and type using WordPerfect on my own. In high school, I also learned how to use keyboard shortcuts like Alt-F4 to exit and F7 to print WordPerfect. When I moved to the United States, I had more freedom to use computers, navigate the internet, and learn anything from YouTube. I joined this wonderful Refcode community to learn more about Software Web Development.
